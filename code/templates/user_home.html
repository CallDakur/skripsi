Website ini merupakan salah satu sarana informasi kesehatan terutama penyakit Pneumonia pada Balita di Kabupaten Bangkalan, Jawa Timur.
Sistem Informasi Geografis (SIG) penyebaran penyakit Pneumonia pada Balita, selain itu pasien bisa melihat penyebaran pasien Pneumonia di Kabupaten Bangkalan dan dapat mendaftar terlebih dahulu untuk melakukan pengecekan kesehatan untuk Balita.
